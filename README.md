homebrew-gds
============

A private Homebrew tap for the [gds-cli](https://github.com/alphagov/gds-cli).

## Pre-requisites

- An SSH key configured, as the repo clones over SSH.

## Usage

```
brew tap alphagov/gds
brew install gds-cli
gds --version
```

or

```
brew install alphagov/gds/gds-cli
gds --version
```
